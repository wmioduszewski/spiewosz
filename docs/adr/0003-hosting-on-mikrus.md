# ADR 0003 – Hosting na VPS Mikrus

**Status:** Accepted

## Context
Projekt ma służyć nauce infrastruktury, reverse proxy, DNS, certyfikatów oraz DevOps.

## Decision
Hosting odbywa się na **VPS Mikrus**, który zapewnia niski koszt i pełną kontrolę nad środowiskiem.

## Alternatives
- Azure App Service — za dużo „magii”.
- Azure Container Apps — podobnie.

## Consequences
- konieczne manualne zarządzanie procesami,
- nauka reverse proxy i sieci.
