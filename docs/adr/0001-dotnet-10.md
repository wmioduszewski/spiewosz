# ADR 0001 – Backend Platform: .NET 10

**Status:** Accepted  
**Date:** 2025-02

## Context
Projekt Spiewosz służy jako poligon doświadczalny dla nowoczesnych technologii backendowych i devopsowych. Autor zawodowo pracuje w .NET, więc naturalne było wybranie technologii zgodnej z codziennym doświadczeniem.

## Decision
Backend projektu będzie implementowany w **.NET 10**, z wykorzystaniem ASP.NET (Minimal API lub MVC).

## Alternatives
- Node.js — odrzucone: mniejsza wartość edukacyjna.
- Go — odrzucone: brak bezpośredniej przydatności zawodowej.

## Consequences
- łatwa konteneryzacja,
- zgodność z Aspire w przyszłości,
- możliwość integracji z Application Insights.
